## 摘要
从单张图像生成铰接3D物体的关键挑战在于准确预测潜在的运动学结构。现有方法要么直接从缺乏动态部件级运动学关系的静态图像中推断运动学参数，要么从单张图像生成的视觉动态中估计参数，这容易导致两个步骤的累积误差。此外，现有标注数据集的规模和多样性有限，进一步阻碍了对复杂真实世界物体的泛化。为了克服这些限制，我们提出学习视觉动态和运动学参数的联合分布。认识到铰接物体可以表示为动态系统，我们提出了一个统一的部分世界模型，称为PWM-ArtGen。为了利用未标注数据，该模型将动作扩散和图像扩散与独立扩散时间步长耦合，使得视觉分支能够进行协同训练。我们还精心构建了一个包含19.7k个部件级图像对的光真实数据集（不含运动学注释），以支持协同训练。实验表明，PWM-ArtGen在静态状态下显著优于现有基线，并对分布外物体展现出强大的零样本泛化能力。

## Abstract
The key challenge in articulated 3D object generation from a single image is accurately predicting the underlying kinematic structure. Existing methods either infer kinematic parameters directly from a static image that lacks dynamic part-level kinematic relationships, or estimate parameters from visual dynamics generated from a single image, which is prone to accumulated errors of two steps. Moreover, the limited scale and diversity of existing annotated datasets further hinder generalization to complex, real-world objects. To overcome these limitations, we propose to learn the joint distribution of visual dynamics and kinematic parameters. Recognizing that articulated objects can be formulated as dynamic systems, we propose a unified Part World Model called PWM-ArtGen. To leverage unannotated data, this model couples action diffusion and image diffusion with independent diffusion timesteps, which enables visual branch co-training. We further curate a photorealistic dataset of 19.7k part-level image pairs without kinematic annotations, to support co-training. Experiments demonstrate that PWM-ArtGen substantially outperforms existing baselines in the resting state and exhibits strong zero-shot generalization to out-of-distribution objects.

## 2. 论文提出的方法论：核心思想、关键技术细节
- **核心思想**：构建一个统一的**部分世界模型（Part World Model）PWM-ArtGen**，基于扩散Transformer联合建模部件的后动作图像（visual dynamics）和运动学参数（kinematic parameters），并采用独立扩散时间步长解耦两个分支，实现在无标注数据上的视觉分支协同训练。
- **关键技术细节**：
  - **数据表示**：每个铰接部件表示为三元组 \((o', a, p_{\text{ref}})\)，其中 \(o'\) 为执行动作后的动态图像（256×256 RGB），\(a\) 为运动学属性（关节类型、运动范围、轴位置、轴方向、部件包围盒、基础包围盒），\(p_{\text{ref}}\) 为部件级结构引导（裁剪部件图像 + 2D包围盒归一化坐标，通过DINOv2编码）。
  - **模型架构**：扩散Transformer，输入为视觉隐码（SDXL VAE编码后patchify）和运动学MLP投影，条件为静态图像全局特征和参考部件嵌入；通过自适应层归一化（AdaLN）注入独立的时间步 \(t_a, t_{o'}\)。
  - **视觉动力学正则化器（VDR）**：借鉴REPA，将扩散过程的中间隐藏特征与DINOv2对干净后动作图像提取的patch特征进行余弦相似度对齐，强制视觉表示与动态一致，减少对静态线索的依赖。
  - **协同训练（Co-Training）**：对有标注数据（含运动学参数）和无标注数据（仅图像对）联合训练。无标注样本将动作扩散步长固定为 \(T\)（全噪声），仅预测图像噪声，平衡因子 \(\gamma=1.0\)。
  - **PM-R数据集**：利用冻结的图像编辑模型（Qwen-Image-Edit-2509）将PartNet-Mobility的合成渲染增强为光真实版本（19.7k图像对），无运动学标注，用于协同训练。

## 8. 不足与局限
- **部件掩码依赖**：在无真实图设定下，性能受限于SAM+GPT-4o的部件分割和连接图预测质量，尤其对合成数据（PM）的渲染伪影敏感，导致dcDist和dCD略差。
- **类别覆盖有限**：训练数据仅来自PartNet-Mobility（存储家具、桌子、冰箱等几类），在ACD及其他真实场景上的泛化可能受限于类别多样性。
- **PM-R的仿真性**：虽然增强为光真实，仍是基于合成渲染的编辑结果，与真实拍摄图像仍有分布差异，仅在零样本测试中验证。
- **运动学类型局限**：仅处理旋转和棱柱关节，未涉及更复杂关节（如球关节、螺旋关节）。
- **失败案例分析**：附录显示，当物体基座几何非标准或铰链位置不规则时，检索出的3D模型与预测运动不匹配，导致碰撞或物理不合理。
- **未与更多近期方法对比**：如FreeArt3D、ArtiLatent等2025年方法未涵盖；与DIPO的对比因数据差异仅为近似比较。
